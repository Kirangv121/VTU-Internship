# 📅 April 24 – Cyber Security Learning Log

## 🧩 Concept of Network Segmentation
**Network segmentation** is the process of dividing a large network into smaller, manageable, and secure segments or subnets.

### 🔹 Purpose:
- Improve security by isolating systems
- Control traffic flow between segments
- Enhance overall network efficiency

---

## 🔐 Benefits of Dividing Networks into Smaller Secure Zones

### 🔹 Key Benefits:
- 🛡️ **Enhanced Security** – Limits spread of malware or attacks
- 🚫 **Access Restriction** – Users access only required resources
- 📊 **Better Visibility** – Easier monitoring and logging
- ⚡ **Improved Performance** – Reduced congestion and broadcast traffic
- 🔒 **Data Protection** – Sensitive data can be isolated

---

## ⚠️ Security Considerations When Implementing Segmentation

### 🔹 Important Points:
- 🔐 **Strong Access Controls** – Define clear communication rules
- 🧱 **Firewall Integration** – Secure boundaries between segments
- 🔍 **Continuous Monitoring** – Track traffic between zones
- 🔄 **Regular Updates** – Keep configurations and systems updated
- 🎯 **Least Privilege Principle** – Limit user permissions

### 🔹 Challenges:
- Complexity in configuration
- Risk of incorrect rule setup
- Requires skilled management

---

## 🚫 Introduction to Intrusion Prevention Systems (IPS)
An **Intrusion Prevention System (IPS)** is a proactive security solution that monitors and blocks malicious network activity in real time.

### 🔹 Key Functions:
- 🔍 Detect suspicious traffic patterns
- 🚫 Block unauthorized access attempts
- 📊 Analyze network behavior
- 🔐 Prevent exploitation of vulnerabilities

### 🔹 Types:
- **Network-based IPS (NIPS)**
- **Host-based IPS (HIPS)**
- **Wireless IPS (WIPS)**

---

## ⚖️ IDS vs IPS Comparison

| Feature | IDS 🔍 | IPS 🚫 |
|--------|--------|--------|
| Function | Detection only | Detection + Prevention |
| Response | Alerts admin | Automatically blocks threats |
| Deployment | Passive | Inline (active) |

---

## ✅ Summary
- Network segmentation enhances security by isolating parts of a network
- It improves performance and limits attack spread
- Proper planning and monitoring are essential
- IPS systems actively prevent threats in real-time
