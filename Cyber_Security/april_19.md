# 📅 April 19 – Cyber Security Learning Log

## ⚠️ Types of Network Attacks

### 🦠 Malware Attacks
**Malware attacks** involve malicious software designed to damage or gain unauthorized access to systems.

#### 🔹 Types:
- Virus
- Worm
- Trojan Horse
- Ransomware
- Spyware

#### 🔹 Impact:
- Data theft or corruption
- System slowdown or failure
- Unauthorized access

---

### 🎭 Man-in-the-Middle (MITM) Attacks
In a **MITM attack**, an attacker secretly intercepts and possibly alters communication between two parties.

#### 🔹 How It Works:
- Attacker positions between sender and receiver
- Intercepts and modifies data

#### 🔹 Impact:
- Data theft
- Credential compromise
- Privacy violation

---

### 🚫 Denial-of-Service (DoS) Attacks
A **DoS attack** overwhelms a system or network, making it unavailable to users.

#### 🔹 Characteristics:
- Flooding network with traffic
- Resource exhaustion

#### 🔹 Impact:
- Service downtime
- Loss of productivity
- Financial damage

---

## 📊 Monitoring Network Traffic
Monitoring network traffic helps detect unusual activity and maintain network performance.

### 🔹 Key Activities:
- Tracking incoming and outgoing data
- Identifying suspicious patterns
- Monitoring bandwidth usage
- Detecting unauthorized access

---

## 🛠️ Using Network Analysis Tools

### 🔍 Wireshark
- Captures and analyzes network packets in real time
- Helps identify network issues and security threats

---

### 🧪 NetworkMiner
- Network forensic analysis tool
- Extracts files, credentials, and data from captured traffic

---

## 🔐 Restricting and Monitoring Network Access

### 🔹 Methods:
- 🔑 **Access Control Lists (ACLs)** – Define who can access network resources
- 🔐 **Authentication Mechanisms** – Ensure only authorized users connect
- 📊 **Logging & Monitoring** – Track user activity
- 🛡️ **Firewalls** – Filter incoming and outgoing traffic
- 🚫 **Network Segmentation** – Separate sensitive areas of the network

---

## ✅ Summary
- Network attacks include malware, MITM, and DoS attacks
- Monitoring traffic helps detect and prevent threats
- Tools like Wireshark and NetworkMiner assist in analysis
- Restricting access improves network security and control
