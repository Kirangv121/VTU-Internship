# 📅 April 15 – Cyber Security Learning Log

## 🔐 Securing Wireless Networks
Securing wireless networks is essential to prevent unauthorized access and protect sensitive data.

### 🔹 Key Measures:
- 🔑 **Use Strong Encryption** – WPA3 or WPA2 security protocols
- 🔐 **Set Strong Passwords** – Avoid default credentials
- 🚫 **Disable SSID Broadcasting** (optional)
- 📍 **MAC Address Filtering** – Allow only trusted devices
- 🔄 **Regular Firmware Updates** – Fix vulnerabilities
- 🛡️ **Enable Firewalls** – Protect against external threats

---

## ☁️ Managing Networks Using Cloud-Based Tools
Cloud-based tools allow remote monitoring and management of networks.

### 🔹 Features:
- 🌍 **Remote Access** – Manage networks from anywhere
- 📊 **Centralized Dashboard** – Monitor multiple devices
- 🔄 **Automatic Updates** – Keep systems secure
- 📈 **Scalability** – Easily expand network infrastructure

### 🔹 Benefits:
- Reduces manual effort
- Improves efficiency
- Enhances security management

---

## ⚠️ Risks Associated with Unsecured Wireless Networks

### 🔹 Common Risks:
- 🕵️ **Unauthorized Access** – Hackers can connect to the network
- 📡 **Data Interception** – Sensitive data can be captured
- 🦠 **Malware Attacks** – Spread through unsecured connections
- 🚫 **Network Misuse** – Unauthorized users consume bandwidth
- 🎭 **Identity Theft** – Personal information can be stolen

---

## 📱 Best Practices for Wireless Device Management

### 🔹 Guidelines:
- 🔐 Use strong passwords and enable **multi-factor authentication**
- 🔄 Keep devices updated with latest security patches
- 📍 Disable unused wireless features (Bluetooth, Wi-Fi when not needed)
- 🛡️ Install antivirus and security software
- 🚫 Avoid connecting to unknown or public Wi-Fi networks
- 📊 Monitor connected devices regularly

---

## ✅ Summary
- Wireless networks must be secured using encryption and strong configurations
- Cloud-based tools simplify network management and improve scalability
- Unsecured networks pose serious risks like data theft and unauthorized access
- Proper device management ensures overall network security and reliability
