# 📅 April 21 – Cyber Security Learning Log

## 🔥 Role of Firewalls in Network Protection
A **firewall** acts as a barrier between a trusted internal network and untrusted external networks (like the internet).

### 🔹 Key Functions:
- 🚫 Blocks unauthorized access
- 🔍 Filters traffic based on rules (IP address, ports, protocols)
- 🛡️ Protects against malware and attacks
- 📊 Logs and monitors network activity

### 🔹 Types of Firewalls:
- **Packet Filtering Firewall** – Filters packets based on rules
- **Stateful Firewall** – Tracks active connections
- **Application Layer Firewall** – Filters at application level
- **Next-Generation Firewall (NGFW)** – Advanced security features like intrusion prevention

---

## 🔐 Understanding Virtual Private Networks (VPNs)
A **VPN (Virtual Private Network)** creates a secure tunnel between a user and the internet.

### 🔹 How It Works:
- Encrypts data before sending
- Routes traffic through a VPN server
- Masks the user’s IP address

### 🔹 Benefits:
- 🔒 Ensures data privacy
- 🌍 Enables secure remote access
- 🕵️ Prevents tracking and eavesdropping
- 🌐 Helps access restricted resources

---

## ✅ Verifying Encryption of VPN Traffic

### 🔹 Methods:
- 🔍 **Packet Inspection** – Use tools to confirm encrypted packets
- 🔐 **Check Protocols** – Ensure use of secure protocols (OpenVPN, IPSec, WireGuard)
- 🌐 **IP Leak Test** – Confirm real IP is hidden
- 🔒 **Browser Indicators** – Look for HTTPS and secure connections

### 🔹 Signs of Proper Encryption:
- Data appears unreadable (ciphertext)
- No exposure of original IP address
- Secure tunneling protocols in use

---

## 📊 Monitoring Network Security Using Analysis Tools

### 🔹 Common Tools:
- 🔍 **Wireshark** – Captures and analyzes packets
- 🚨 **Snort** – Intrusion detection/prevention system
- 📊 **Nagios** – Network monitoring and alerts
- 🧪 **NetworkMiner** – Network forensic analysis

### 🔹 Key Uses:
- Detect suspicious or malicious activity
- Monitor bandwidth and traffic patterns
- Identify vulnerabilities and threats
- Ensure network performance and security

---

## ✅ Summary
- Firewalls control and secure network traffic
- VPNs provide encrypted and private communication
- Encryption verification ensures VPN security
- Network analysis tools help monitor and protect networks
