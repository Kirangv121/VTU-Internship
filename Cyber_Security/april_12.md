# 📅 April 12 – Cyber Security Learning Log

## 🔐 Importance of Strong Authentication Methods
Authentication verifies the identity of users before granting access to systems or networks.

### 🔹 Why It Is Important:
- Prevents unauthorized access
- Protects sensitive data
- Ensures accountability of users
- Reduces risk of cyberattacks

### 🔹 Types of Authentication:
- **Something you know** (password, PIN)
- **Something you have** (OTP, security token)
- **Something you are** (biometrics)

---

## 🔑 Password Security Best Practices

### 🔹 Guidelines:
- Use **strong and complex passwords** (mix of letters, numbers, symbols)
- Avoid common or easily guessable passwords
- Use **unique passwords** for different accounts
- Change passwords regularly
- Enable **multi-factor authentication (MFA)**
- Do not share passwords with others

### 🔹 Example of Strong Password:
- `G7@kL9!pQ#2`

---

## 🧬 Biometric Authentication
Biometric authentication uses unique physical or behavioral characteristics to verify identity.

### 🔹 Types:
- Fingerprint recognition
- Facial recognition
- Iris scanning
- Voice recognition

### 🔹 Advantages:
- High security
- Difficult to replicate
- Convenient for users

### 🔹 Limitations:
- Privacy concerns
- Requires specialized hardware
- May have false positives/negatives

---

## 🛡️ Role of Antivirus Software
Antivirus software protects systems from malware and other threats.

### 🔹 Functions:
- Detects and removes viruses, worms, trojans
- Provides real-time protection
- Scans files and downloads
- Prevents malicious activities

### 🔹 Benefits:
- Improves system security
- Protects data integrity
- Reduces risk of cyberattacks

---

## 🔄 Software Updates and Security Patches

### 🔹 Importance:
- Fix known vulnerabilities
- Improve system performance
- Enhance security features

### 🔹 Best Practices:
- Enable automatic updates
- Regularly check for updates
- Install patches as soon as available

---

## ✅ Summary
- Strong authentication ensures only authorized access
- Good password practices enhance security
- Biometrics provide advanced authentication methods
- Antivirus software protects against malware
- Regular updates and patches fix vulnerabilities and improve security
