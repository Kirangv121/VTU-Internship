# 📅 April 28 – Cyber Security Learning Log

## 🛡️ Integrating Malware Protection, Network Security, and Data Protection
Effective cybersecurity requires combining multiple security measures to protect systems, networks, and data.

### 🔹 Key Components:
- 🦠 **Malware Protection** – Antivirus and anti-malware tools to detect and remove threats
- 🌐 **Network Security** – Firewalls, IDS/IPS to secure network traffic
- 🔐 **Data Protection** – Encryption, access control, and DLP systems

### 🔹 Integration Benefits:
- Provides comprehensive security coverage
- Reduces risk of single point of failure
- Enhances threat detection and response

---

## 🧱 Creating Layered Security Architecture
A **layered security architecture** (Defense in Depth) uses multiple layers of security controls to protect systems.

### 🔹 Layers Include:
1. **Physical Security** – Protect hardware and infrastructure  
2. **Network Security** – Firewalls, segmentation  
3. **Endpoint Security** – Device protection  
4. **Application Security** – Secure software and services  
5. **Data Security** – Encryption and access control  

### 🔹 Advantages:
- Multiple barriers against attacks
- Increased resilience
- Improved detection and prevention

---

## 🚨 Monitoring and Responding to Security Incidents

### 🔹 Monitoring:
- 📊 Continuous tracking of network and system activity
- 🔍 Use of security tools (SIEM, IDS/IPS)
- 📈 Log analysis and alert systems

### 🔹 Incident Response:
1. **Identification** – Detect and confirm the incident  
2. **Containment** – Limit the spread of the attack  
3. **Eradication** – Remove the threat  
4. **Recovery** – Restore systems and services  
5. **Lessons Learned** – Improve future security measures  

---

## ✅ Summary
- Integrating multiple security domains strengthens overall protection
- Layered security ensures multiple defense mechanisms
- Continuous monitoring and quick response are critical to minimizing damage from security incidents
