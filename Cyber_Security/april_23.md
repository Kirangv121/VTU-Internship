# 📅 April 23 – Cyber Security Learning Log

## 🧩 Concept of Network Segmentation
**Network segmentation** is the practice of dividing a large network into smaller, isolated sections called **segments** or **subnets**.

### 🔹 Purpose:
- Improve security
- Enhance performance
- Simplify network management

### 🔹 How It Works:
- Each segment acts as an independent network
- Communication between segments is controlled using security policies

---

## 🔐 Benefits of Dividing Networks into Smaller Secure Zones

### 🔹 Key Advantages:
- 🛡️ **Improved Security** – Limits spread of attacks within network
- 🚫 **Access Control** – Restricts user access to specific segments
- 📊 **Better Monitoring** – Easier to track traffic within segments
- ⚡ **Enhanced Performance** – Reduces network congestion
- 🔍 **Threat Containment** – Isolates infected systems quickly

---

## ⚠️ Security Considerations When Implementing Segmentation

### 🔹 Important Factors:
- 🔐 **Define Access Policies** – Control communication between segments
- 🧱 **Use Firewalls** – Protect boundaries between segments
- 🔍 **Monitor Traffic** – Detect suspicious activity between zones
- 🔄 **Regular Updates** – Keep systems and rules updated
- 🎯 **Least Privilege Principle** – Grant only necessary access

### 🔹 Challenges:
- Increased complexity in management
- Need for proper configuration
- Risk of misconfiguration leading to vulnerabilities

---

## 🚫 Introduction to Intrusion Prevention Systems (IPS)
An **Intrusion Prevention System (IPS)** is a network security tool that monitors traffic and actively blocks malicious activities.

### 🔹 Functions:
- 🔍 Detects suspicious patterns
- 🚫 Blocks threats in real-time
- 📊 Monitors network traffic
- 🔐 Prevents attacks like DoS and malware spread

### 🔹 Types of IPS:
- **Network-based IPS (NIPS)**
- **Host-based IPS (HIPS)**
- **Wireless IPS (WIPS)**

---

## ⚖️ IDS vs IPS (Quick View)

| Feature | IDS 🔍 | IPS 🚫 |
|--------|--------|--------|
| Action | Detects threats | Detects + Prevents |
| Response | Alerts only | Blocks attacks |
| Placement | Passive | Inline (active) |

---

## ✅ Summary
- Network segmentation divides networks into secure zones
- It improves security, performance, and control
- Proper planning is required to avoid misconfiguration risks
- IPS systems actively detect and prevent network attacks
