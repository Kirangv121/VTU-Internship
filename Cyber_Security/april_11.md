# 📅 April 11 – Cyber Security Learning Log

## ⚠️ Understanding Network Security Threats
A **network security threat** is any activity or event that can harm, disrupt, or gain unauthorized access to a network and its data.

### 🔹 Examples:
- Unauthorized access
- Data theft
- Malware infections
- Service disruptions

---

## 🔍 Difference Between Key Terms

### 🧨 Threat
A **threat** is a potential danger that can exploit a system.

- Example: A hacker attempting to break into a network

---

### 🕳️ Vulnerability
A **vulnerability** is a weakness in a system that can be exploited.

- Example: Weak passwords, outdated software

---

### ⚡ Risk
A **risk** is the likelihood that a threat will exploit a vulnerability and cause damage.

- Example: High risk if a system has weak security and is exposed to the internet

---

### 🦠 Malware
**Malware** (malicious software) is software designed to harm or exploit systems.

#### 🔹 Types:
- Virus
- Worm
- Trojan Horse
- Ransomware
- Spyware

---

## 🔄 Active vs Passive Attacks

### 🔴 Active Attacks
- Directly affect system operation
- Modify or disrupt data

#### 🔹 Examples:
- Denial of Service (DoS)
- Data modification
- Man-in-the-Middle attacks

---

### 🟢 Passive Attacks
- Do not affect system directly
- Focus on monitoring or stealing information

#### 🔹 Examples:
- Eavesdropping
- Traffic analysis

---

## 🛠️ Methods for Identifying Security Threats

### 🔹 Techniques:
- 📊 **Network Monitoring** – Observing traffic patterns
- 🧾 **Log Analysis** – Reviewing system and network logs
- 🚨 **Intrusion Detection Systems (IDS)** – Detect suspicious activities
- 🔍 **Vulnerability Scanning** – Identifying system weaknesses
- 🧠 **Behavior Analysis** – Detect unusual user or system behavior

---

## ✅ Summary
- Threats are potential dangers, vulnerabilities are weaknesses, and risk is the impact likelihood
- Malware is harmful software targeting systems
- Active attacks disrupt systems, while passive attacks silently monitor
- Identifying threats involves monitoring, analysis, and security tools
