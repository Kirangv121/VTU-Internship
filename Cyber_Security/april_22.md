# 📅 April 22 – Cyber Security Learning Log

## 🦠 Understanding Malware Threats
**Malware (Malicious Software)** refers to programs designed to harm, exploit, or gain unauthorized access to systems.

---

### 🧬 Viruses
A **virus** is a type of malware that attaches itself to legitimate programs and spreads when the program is executed.

#### 🔹 Characteristics:
- Requires user action to spread
- Infects files and programs
- Can corrupt or delete data

---

### 🐛 Worms
A **worm** is a self-replicating malware that spreads automatically across networks.

#### 🔹 Characteristics:
- Does not require user interaction
- Consumes network bandwidth
- Spreads rapidly

---

### 🐴 Trojans
A **Trojan Horse** is malware disguised as legitimate software.

#### 🔹 Characteristics:
- Tricks users into installing it
- Creates backdoors for attackers
- Steals sensitive information

---

### 🔐 Ransomware
**Ransomware** encrypts a user's data and demands payment to restore access.

#### 🔹 Characteristics:
- Locks files or systems
- Demands ransom (usually in cryptocurrency)
- Can cause major data loss

---

## 🛡️ Role of Anti-Virus and Anti-Malware Software

### 🔹 Functions:
- 🔍 Detects malicious software
- 🧹 Removes or quarantines threats
- 🛡️ Provides real-time protection
- 🔄 Updates virus definitions regularly

### 🔹 Benefits:
- Protects system integrity
- Prevents data loss
- Reduces risk of cyberattacks

---

## ⚡ Real-Time Protection vs On-Demand Scanning

### 🟢 Real-Time Protection
- Continuously monitors system activity
- Detects and blocks threats instantly

### 🔵 On-Demand Scanning
- Manual or scheduled scans
- Checks system for existing threats

### 🔹 Difference:

| Feature              | Real-Time Protection 🟢 | On-Demand Scanning 🔵 |
|----------------------|------------------------|----------------------|
| Monitoring           | Continuous             | Manual/Scheduled     |
| Response Time        | Immediate              | Delayed              |
| Resource Usage       | Moderate               | High during scan     |

---

## 🤖 Using AI and Machine Learning for Threat Detection

### 🔹 How It Works:
- Analyzes patterns in network and system behavior
- Detects anomalies and unusual activities
- Learns from previous threats to improve detection

### 🔹 Advantages:
- Faster threat detection
- Identifies unknown (zero-day) attacks
- Reduces human effort
- Improves accuracy over time

---

## ✅ Summary
- Malware includes viruses, worms, trojans, and ransomware
- Antivirus software helps detect and remove threats
- Real-time protection provides continuous security, while on-demand scanning checks systems manually
- AI and machine learning enhance modern threat detection capabilities
