# 📅 April 10 – Cyber Security Learning Log

## ⚠️ Common WLAN Security Threats
Wireless LANs (WLANs) are vulnerable to various security threats due to their open-air communication.

### 🔹 Common Threats:
- 🕵️ **Eavesdropping** – Unauthorized users intercept network traffic
- 🎭 **Man-in-the-Middle (MITM) Attacks** – Attacker intercepts and alters communication
- 📡 **Rogue Access Points** – Unauthorized APs set up to trick users
- 🔓 **Weak Encryption Attacks** – Exploiting weak security protocols (WEP, WPA)
- 🚫 **Denial of Service (DoS)** – Flooding network to disrupt services
- 🔑 **Password Cracking** – Guessing or brute-forcing Wi-Fi passwords

---

## 🛠️ Wireless Protocol Analysis Tools
These tools help analyze and troubleshoot wireless network traffic.

### 🔹 Common Tools:
- 📊 **Wireshark** – Captures and analyzes network packets
- 📡 **Aircrack-ng** – Tests Wi-Fi security and password strength
- 📶 **Kismet** – Detects networks and monitors traffic
- 🔍 **NetSpot** – Wi-Fi analysis and site survey tool

### 🔹 Uses:
- Monitor network performance
- Detect suspicious activity
- Identify vulnerabilities

---

## 📡 Monitoring Wireless Network Activity

Monitoring helps in identifying unusual behavior and maintaining network health.

### 🔹 Key Activities:
- 📊 Tracking bandwidth usage
- 👥 Monitoring connected devices
- 🚨 Detecting unauthorized access
- 📈 Analyzing traffic patterns

### 🔹 Methods:
- Real-time monitoring dashboards
- Log analysis
- Intrusion detection systems (IDS)

---

## 🔐 Protecting Enterprise Wireless Networks from Attacks

### 🔹 Security Measures:
- 🔑 **Strong Encryption** – Use WPA3 or WPA2
- 🔐 **Secure Authentication** – Implement multi-factor authentication (MFA)
- 🚫 **Disable SSID Broadcasting** (optional for added security)
- 📍 **MAC Address Filtering** – Allow only authorized devices
- 🔄 **Regular Updates** – Keep firmware and software updated
- 🛡️ **Firewalls & IDS/IPS** – Detect and prevent attacks

### 🔹 Best Practices:
- Conduct regular security audits
- Use VPNs for secure remote access
- Segment networks (guest vs internal)
- Train users on security awareness

---

## ✅ Summary
- WLANs face threats like eavesdropping, MITM, and rogue access points
- Protocol analysis tools help detect vulnerabilities and monitor traffic
- Continuous monitoring ensures network safety and performance
- Strong encryption and security practices protect enterprise wireless networks
