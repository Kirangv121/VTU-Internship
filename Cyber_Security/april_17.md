# 📅 April 17 – Cyber Security Learning Log

## 📱 Device Security Techniques
Device security focuses on protecting computers, laptops, and mobile devices from unauthorized access and threats.

### 🔹 Key Techniques:
- 🔐 **Enable Device Lock** – Use PIN, password, or biometric locks
- 🛡️ **Install Antivirus/Anti-malware** – Protect against threats
- 🔄 **Regular Updates** – Keep OS and apps updated
- 🔒 **Disk Encryption** – Protect stored data
- 🚫 **Disable Unused Ports & Services** – Reduce attack surface
- 📍 **Use Firewalls** – Block unauthorized access

---

## 🔑 Password Protection and Authentication

### 🔹 Best Practices:
- Use **strong and complex passwords**
- Avoid reusing passwords across multiple accounts
- Enable **Multi-Factor Authentication (MFA)**
- Change passwords periodically
- Use password managers for secure storage

### 🔹 Authentication Methods:
- **Knowledge-based** (passwords, PINs)
- **Possession-based** (OTP, tokens)
- **Biometric-based** (fingerprint, face recognition)

---

## ☁️ Backup Strategies Using Cloud Services

Cloud backups ensure data is safe and recoverable in case of device failure or cyberattacks.

### 🔹 Strategies:
- 🔄 **Regular Backups** – Schedule automatic backups
- ☁️ **Cloud Storage Services** – Store data remotely
- 📂 **Incremental Backups** – Backup only changed data
- 🔐 **Encrypted Backups** – Protect backup data

### 🔹 Benefits:
- Data recovery during system failures
- Protection against ransomware attacks
- Accessibility from anywhere

---

## 📍 Device Tracking and Monitoring Tools

### 🍎 macOS
- **Find My Mac** – Locate lost or stolen devices
- **Activity Monitor** – Monitor system performance
- **FileVault** – Full disk encryption

---

### 🐧 Linux
- **Top / Htop** – Monitor system processes
- **Log Files** – Track system activities
- **Fail2Ban** – Prevent unauthorized access attempts

---

### 🪟 Windows
- **Find My Device** – Track device location
- **Windows Defender** – Built-in security protection
- **Event Viewer** – Monitor system logs and events

---

## ✅ Summary
- Device security involves protecting systems using multiple techniques
- Strong password practices and authentication enhance security
- Cloud backups ensure data safety and recovery
- Monitoring tools help track and secure devices across macOS, Linux, and Windows
