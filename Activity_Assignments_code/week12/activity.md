## 🧩 Concept of Network Segmentation
Network segmentation is the process of dividing a large network into smaller, isolated sections to improve security and management.

---

## 🔐 Benefits of Network Segmentation

### 🔹 Security
- Limits spread of cyberattacks
- Protects sensitive data

### 🔹 Control
- Restricts access between segments
- Improves policy enforcement

### 🔹 Performance
- Reduces network congestion
- Improves efficiency
