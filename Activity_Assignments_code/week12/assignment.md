## 🧩 Network Segmentation Design

### 🔹 Segmentation Plan:
- Divide network into multiple zones:
  - 🏢 **Internal Network (Employees)**
  - 🌐 **Guest Network**
  - 🗄️ **Server Network**
  - 🔐 **Sensitive Data Zone**

---

### 🔹 Access Control:
- Restrict communication between segments
- Apply **least privilege principle**
- Use firewalls between segments

---

## 🚫 IPS Implementation Plan

### 🔹 IPS Placement:
- Deploy IPS **between network segments**
- Place IPS at:
  - Network perimeter
  - Between critical zones

---

### 🔹 IPS Functions:
- Monitor network traffic
- Detect malicious activity
- Block threats in real-time

---

### 🔹 Types of IPS:
- Network-based IPS (NIPS)
- Host-based IPS (HIPS)
