## 🧩 Network Setup Plan

### 📡 Network Design
- Use a **centralized wireless architecture**
- Deploy **multiple access points (APs)** for full coverage
- Ensure proper **AP placement** to avoid dead zones
- Use **dual-band routers (2.4 GHz & 5 GHz)**

---

### 🌐 IP Addressing
- Use **DHCP** for automatic IP assignment
- Segment network into:
  - Internal network
  - Guest network

---

### 🔐 Security Strategy

#### 🔹 Encryption
- Use **WPA3 (preferred)** or **WPA2**
- Enable strong password policies

#### 🔹 Access Control
- MAC address filtering
- Role-based access (admin, user, guest)

#### 🔹 Network Protection
- Enable firewall
- Use IDS/IPS systems
- Disable unused ports and services

#### 🔹 Monitoring
- Use network monitoring tools
- Track connected devices and traffic
