# 📅 Week 10 Assignment – Cyber Security

## 📝 Title: Network Security Risk Assessment

## 📖 Description
This assignment focuses on identifying vulnerabilities in a network, evaluating associated risks, and proposing effective mitigation strategies to enhance overall security.

---

## 🛠️ Methodologies
- Risk analysis
- Scenario-based evaluation
- Strategic planning

---

## 🎯 Objectives
- Assess network vulnerabilities
- Evaluate potential risks
- Propose mitigation strategies

---

## 📚 Reference / Source Code
- Risk management frameworks (ISO 27001, NIST)
- Cybersecurity best practices

---

## 🔍 Risk Assessment Report

### 🔹 Step 1: Identify Assets
- Network devices (routers, switches, servers)
- Data (sensitive and non-sensitive)
- Users and endpoints

---

### 🔹 Step 2: Identify Vulnerabilities
- Weak passwords
- Outdated software
- Open ports
- Lack of encryption
- Misconfigured firewalls

---

### 🔹 Step 3: Identify Threats
- Malware attacks
- Unauthorized access
- Phishing attacks
- Denial-of-Service (DoS)

---

### 🔹 Step 4: Risk Evaluation

| Threat              | Vulnerability        | Impact        | Risk Level |
|--------------------|---------------------|--------------|-----------|
| Data breach        | Weak authentication | High         | High      |
| Malware infection  | No antivirus        | Medium       | Medium    |
| DoS attack         | No firewall         | High         | High      |

---

## 🛡️ Mitigation Plan

### 🔐 Security Measures:
- Implement **strong authentication (MFA)**
- Use **WPA3/WPA2 encryption**
- Install and update **antivirus software**
- Configure **firewalls and IDS/IPS**

---

### 🔄 System Management:
- Regular software updates and patches
- Continuous network monitoring
- Log analysis and auditing

---

### 👨‍🏫 User Awareness:
- Conduct security training
- Educate users about phishing and threats

---

- Risk assessment report
- Mitigation strategy plan
