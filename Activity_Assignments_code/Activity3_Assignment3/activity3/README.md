# Activity 3 – Create React App Setup Using Vite

## Overview
This activity focuses on setting up a React development environment using **Vite**, a modern and fast frontend build tool. Vite helps developers create and run React applications quickly with minimal configuration.

Unlike traditional tools, Vite provides **faster startup time and efficient development performance**.

In this task, a React project is initialized using the command `npm create vite@latest`. After creating the project, the required dependencies are installed and the development server is started. Students then explore the generated project structure to understand the purpose of different folders and files.

This activity helps students understand how a React project is structured and where the main application code is written. Learning the folder structure is important because it helps developers organize components, styles, and logic properly when building large applications.

---

# Steps to Setup React Project Using Vite

## Step 1: Install Node.js
Before creating a React project, **Node.js must be installed** because React uses Node and npm (Node Package Manager) to manage dependencies and run development tools.

Download Node.js from the official website and install it.

---

## Step 2: Create a React Project

Open the terminal or command prompt and run the following command:

```bash
npm create vite@latest activity3
```

During setup choose:

- **Project Name:** activity3  
- **Framework:** React  
- **Variant:** JavaScript  

---

## Step 3: Move into the Project Folder

```bash
cd activity3
```

---

## Step 4: Install Dependencies

```bash
npm install
```

---

## Step 5: Start the React Application

```bash
npm run dev
```

This command starts the development server and opens the application in the browser at:

```
http://localhost:5173/
```

---

# React Folder Structure

Below is the typical folder structure generated by Vite for a React project:

```
activity3
│
├── node_modules
├── public
├── src
│   ├── App.jsx
│   ├── main.jsx
│   └── assets
│
├── package.json
├── vite.config.js
└── README.md
```

---

# React Folder Structure Explanation

### node_modules
This folder contains all the installed libraries and dependencies required for the React application.

### public
This folder contains static files such as images or icons that are directly served by the browser.

### src
This is the main folder where React code is written. Components, CSS files, and application logic are placed here.

### main.jsx
This file is the **entry point** of the React application. It renders the main **App component** to the browser.

### App.jsx
This is the **main component** where the main user interface of the application is built.

### package.json
This file contains project metadata and lists all dependencies used in the project.

### vite.config.js
This file contains configuration settings for the Vite build tool.
