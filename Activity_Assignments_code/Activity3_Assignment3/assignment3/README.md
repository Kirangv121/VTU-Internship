# Assignment 3 – Dynamic Content Mini Project

## Overview
The objective of this assignment is to create a simple **React interface** that displays **dynamic content using state variables**.

In React, dynamic content is handled using the **useState hook**, which allows components to store and update data. When the state changes, React automatically updates the **user interface (UI)**.

This mini project demonstrates how React components can display and update dynamic data efficiently.

---

## Project Description
In this mini project, a component is created to display **user profile information** such as:

- Name
- Role
- Location
- Skills
- Experience


The data is stored inside a **state variable** and displayed using **JSX** within the React component.

The component is created inside the **components folder** to maintain proper **project structure and separation of concerns**. This approach makes the code **organized, reusable, and easy to maintain**.
