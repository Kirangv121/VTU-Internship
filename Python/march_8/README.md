# Python Basics – Introduction

## 1. What is Python?
Python is a **high-level, general-purpose programming language** used to build software, automate tasks, analyze data, and develop applications. It is known for its **simple syntax and readability**, which makes it beginner-friendly and widely used by developers.

---

## 2. History of Python
Python was created by **Guido van Rossum** and first released in **1991**. The goal was to design a language that is **easy to read, easy to write, and powerful enough for complex tasks**. Over time, Python has evolved with versions like **Python 2** and **Python 3**, with Python 3 being the modern standard used today.

---

## 3. Why Python is Popular
Python is popular because it focuses on **simplicity, productivity, and versatility**. Its clean syntax allows developers to write fewer lines of code compared to many other languages. Python also has a **huge ecosystem of libraries and frameworks**, making it useful in many fields such as web development, data science, AI, and automation.

---

## 4. Installing Python
To start using Python, you need to install it on your computer.

Steps:
1. Go to the official Python website.
2. Download the latest version of Python.
3. Run the installer and enable **"Add Python to PATH"**.
4. Complete the installation.
5. Verify installation by running `python --version` in the terminal.

---

## 5. Running Python (IDLE / VS Code / Terminal)

### IDLE
IDLE is Python’s built-in development environment where you can directly write and run Python code.

### VS Code
Visual Studio Code is a popular code editor used by developers. With the **Python extension**, it provides features like debugging, syntax highlighting, and code suggestions.

### Terminal / Command Prompt
You can also run Python programs directly from the terminal using the Python interpreter by typing `python` or running `.py` files.

---

Example: print("Hello, World!")



# Python Features

## 1. Interpreted
Python is an interpreted language where code is executed **line by line by the Python interpreter**, making debugging easier.

## 2. Dynamically Typed
Python automatically determines the **data type of variables at runtime**, so you do not need to declare variable types.

## 3. Object-Oriented
Python supports **object-oriented programming concepts like classes and objects** for building structured and reusable programs.

## 4. Cross-Platform
Python programs can run on **multiple operating systems such as Windows, Linux, and macOS without changing the code**.
