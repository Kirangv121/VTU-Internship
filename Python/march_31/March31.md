# Python Mini Project: Banking Application

---

## 1. Project Overview

The **Banking Application** is a mini project built using Python that simulates basic banking operations.

It uses:
- Functions
- Data Structures
- OOP (Classes & Objects)
- Conditional Logic
- Loops

This system allows users to **create accounts, deposit money, withdraw money, and check balance**.

---

## 2. Features

- Create new account
- Deposit money
- Withdraw money
- Check balance
- Exit system

---

## 3. Concepts Used

### Functions
Used to perform operations like deposit, withdraw, etc.

### Data Structures
- List to store multiple accounts

### OOP
- Class for Bank Account
- Class for Banking System

### Conditional Logic
Used for validation (sufficient balance, account exists)

### Loops
Used for menu-driven program

---

## 4. Account Structure

Each account contains:
- `account_number`
- `name`
- `balance`

---

## 5. Program Flow

1. Display menu
2. Take user input
3. Perform operation
4. Repeat until exit

---

## 6. Sample Menu

```
1. Create Account
2. Deposit Money
3. Withdraw Money
4. Check Balance
5. Exit
```

---

## 7. Conclusion

This project demonstrates how Python concepts can be used to simulate a **real-world banking system**.  
It helps in understanding **OOP design, data handling, and logical decision-making**.  
It is useful for beginners to practice building **interactive console applications**.  
Overall, it strengthens problem-solving and coding skills.
