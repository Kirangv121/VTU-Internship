# Python Final Mini Project: Inventory Management System

---

## 1. Project Overview

The **Inventory Management System** is a mini project built using Python that combines:

- Functions
- Data Structures (Lists & Dictionaries)
- OOP (Classes & Objects)
- Conditional Logic
- Loops

This system allows users to **manage product inventory efficiently**.

---

## 2. Features

- Add new product
- View all products
- Search product by ID
- Update product details
- Delete product
- Exit system

---

## 3. Concepts Used

### Functions
Used to organize different operations like add, view, update, and delete.

### Data Structures
- **List** → to store multiple products  
- **Dictionary** → to store product details (id, name, quantity, price)

### OOP
- Class for Product
- Class for Inventory System

### Conditional Logic
Used for decision making (menu options, searching, updating).

### Loops
Used to repeatedly show menu and iterate through products.

---

## 4. Product Structure

Each product contains:
- `id`
- `name`
- `quantity`
- `price`

---

## 5. Program Flow

1. Display menu
2. Take user input
3. Perform operation
4. Repeat until exit

---

## 6. Sample Menu

```
1. Add Product
2. View Products
3. Search Product
4. Update Product
5. Delete Product
6. Exit
```

---

## 7. Conclusion

This project demonstrates how multiple Python concepts work together to build a **real-world application**.  
It strengthens understanding of **logic building, OOP design, and data handling**.
