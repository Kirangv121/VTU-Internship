# Python Mini Project: Library Management System

---

## 1. Project Overview

The **Library Management System** is a mini project built using Python that helps in managing books in a library.

This project combines:
- Functions
- Data Structures (Lists & Dictionaries)
- OOP (Classes & Objects)
- Conditional Logic
- Loops

It allows users to **add, view, search, issue, return, and delete books**.

---

## 2. Features

- Add new book
- View all books
- Search book by ID
- Issue book
- Return book
- Delete book
- Exit system

---

## 3. Concepts Used

### Functions
Used to organize operations like add, search, issue, return, etc.

### Data Structures
- **List** → to store multiple books  
- **Dictionary / Object attributes** → to store book details

### OOP
- Class for Book
- Class for Library System

### Conditional Logic
Used for checking availability and user choices

### Loops
Used for menu repetition and traversing book records

---

## 4. Book Structure

Each book contains:
- `id`
- `title`
- `author`
- `available` (True/False)

---

## 5. Program Flow

1. Display menu
2. Take user input
3. Perform selected operation
4. Repeat until exit

---

## 6. Sample Menu

```
1. Add Book
2. View Books
3. Search Book
4. Issue Book
5. Return Book
6. Delete Book
7. Exit
```

---

## 7. Conclusion

This project demonstrates how Python concepts can be used to build a **real-world system**.  
It improves understanding of **OOP, logic building, and data management**.
