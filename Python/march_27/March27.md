# Python OOP Project: Student Management System

---

## 1. Project Overview

The **Student Management System** is a simple Python project built using **Object-Oriented Programming (OOP)** concepts.  
It allows users to **add, view, search, update, and delete student records**.

This project demonstrates:
- Classes & Objects
- Encapsulation
- Methods
- List handling

---

## 2. Features

- Add new student
- View all students
- Search student by ID
- Update student details
- Delete student record

---

## 3. Class Design

### Student Class
Represents a single student with attributes:
- `id`
- `name`
- `age`
- `course`

### StudentManagementSystem Class
Handles all operations:
- Add student
- Display students
- Search student
- Update student
- Delete student

---

## 4. Concepts Used

- OOP (Class & Object)
- Encapsulation
- Lists to store objects
- Functions (methods)

---

## 5. Working Flow

1. User selects an option from menu
2. Program performs the selected operation
3. Data is stored in memory using a list
4. Loop continues until user exits

---

## 6. Output Example

```
1. Add Student
2. View Students
3. Search Student
4. Update Student
5. Delete Student
6. Exit
```

---

## 7. Conclusion

This project helps in understanding how **real-world applications are built using OOP concepts** in Python.  
It is a beginner-friendly project to practice **class design and data handling**.
